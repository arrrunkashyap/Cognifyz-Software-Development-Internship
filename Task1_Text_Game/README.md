# 🎮 Treasure Hunt Adventure

A simple and interactive text-based adventure game developed in **C++** as part of the **Cognifyz Technologies Software Development Internship**.

## 📌 Description

In this game, the player explores different locations in search of treasure while facing random events such as monsters and magical crystals. The game tracks the player's health and score throughout the adventure.

---

## 🚀 Features

- 👤 Player name input
- 🛤️ Three different adventure paths
- 🎲 Random game events
- 👹 Monster encounters
- 💎 Magical crystal rewards
- 🎁 Hidden treasure discovery
- ❤️ Health tracking
- ⭐ Score tracking
- 🔁 Play again option
- ✅ Input validation

---

## 🛠️ Technologies Used

- C++
- Object-Oriented Programming Concepts
- Standard Template Library (STL)

## ▶️ How to Run

Compile the program:

```bash
g++ main.cpp -o game
```

Run the executable:

```bash
./game
```

On Windows:

```bash
game.exe
```
