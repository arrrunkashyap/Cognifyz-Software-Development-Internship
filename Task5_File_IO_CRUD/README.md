# 📁 File-Based Task Manager

## Description

A C++ console application that performs CRUD operations on tasks while storing all data in a text file. The program automatically loads saved tasks at startup and saves every change to ensure data persistence.

---

## Features

- Add Task
- View Tasks
- Update Task
- Delete Task
- Search Task
- Mark Completed
- Automatic Save & Load
- Task Statistics
- Sort by ID
- Input Validation

---

## Technologies Used

- C++
- STL Vector
- Structures
- File Handling (`fstream`)
- Algorithms (`sort`)
- Functions

## How to Run

Compile:

```bash
g++ main.cpp -o taskmanager
```

Run:

```bash
taskmanager.exe
```
