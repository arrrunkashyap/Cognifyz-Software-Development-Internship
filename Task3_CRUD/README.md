# 📋 Task Manager (CRUD Console Application)

## 📌 Description

A menu-driven **CRUD (Create, Read, Update, Delete)** application developed in **C++**. The program allows users to manage daily tasks through a simple console interface.

---

## ✨ Features

- ➕ Add Task
- 📋 View All Tasks
- ✏️ Update Existing Task
- ❌ Delete Task
- 🔍 Search Task by ID
- ✅ Mark Task as Completed
- 🆔 Automatic Task IDs
- ✔ Input Validation

---
