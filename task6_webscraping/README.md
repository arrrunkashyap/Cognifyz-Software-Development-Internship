# 🌐 Interactive Web Scraping Tool

## Description

A Python-based web scraping tool that extracts useful information from websites using **Requests** and **BeautifulSoup**.

## Features

- Get webpage title
- Extract headings (H1-H6)
- Extract hyperlinks
- Export links to CSV
- Error handling
- Menu-driven interface

## Technologies Used

- Python
- Requests
- BeautifulSoup4
- Pandas

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python main.py
```
